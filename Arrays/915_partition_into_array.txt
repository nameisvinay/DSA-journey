### âœ… Code (Python):

      ```python
      def partitionDisjoint(nums):
          leftMax = nums[0]
          overallMax = nums[0]
          partitionIdx = 0
          
          for i in range(1, len(nums)):
              overallMax = max(overallMax, nums[i])
              if nums[i] < leftMax:
                  leftMax = overallMax
                  partitionIdx = i
          
          return partitionIdx + 1
      ```
